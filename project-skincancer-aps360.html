Skin cancer detection (APS360)

APS360 · Python · PyTorch · VGG16 · HAM10000 · Google Colab

7-class skin lesion classification on the HAM10000 dataset: reproducible preprocessing, stratified train/val/test splits, augmentation, a Random Forest baseline, and a VGG16 transfer-learning model with a custom classifier head — compared on in-distribution test data and an external ISIC set.

Target classes: akiec, bcc, bkl, df, mel, nv, vasc. The CNN pipeline uses ImageNet-initialized VGG16 with frozen conv blocks and trains only the new head plus fine-tuning strategy as described below.

Project overview

Automated dermoscopic classification can support triage and education, but is not a substitute for clinical diagnosis. This project implements a full CNN workflow with imbalance-aware training and compares against a classical baseline.

Dataset

  • Source: HAM10000 (Kaggle / ISIC 2018)
  • Images: 10,015 dermoscopic images
  • Split: 70% train (7,010) · 15% validation (1,502) · 15% test (1,503), stratified

Class distribution is highly imbalanced: nv dominates (~67%), while vasc (~1.4%) and df (~1.1%) are rare. Oversampling of minority classes was used to reduce bias during training.

Preprocessing

  • Resize / crop to 224×224
  • Normalize with ImageNet mean and standard deviation
  • Augmentation: random horizontal/vertical flips, rotations, color jitter, random greyscale
  • Oversampling of minority classes

Model architecture

  • Backbone: VGG16 with ImageNet weights; convolutional layers frozen
  • Classifier head: Dense(1024)+BN+ReLU+Dropout(0.3) → Dense(512)+BN+ReLU+Dropout(0.2) → Dense(128)+BN+ReLU+Dropout(0.2) → Dense(7, softmax)
  • Trainable parameters: ~26.3M

Baseline model (Random Forest)

  • Validation accuracy: 72.2%
  • Test accuracy: 71.5%
  • F1: strong on majority class nv (~0.85), but 0.00 on rare classes df and vasc

Training (VGG16)

Environment: Google Colab (GPU).

  • Validation accuracy: 74.5%
  • Test accuracy: 74.7%
  • Macro F1: 0.60 (vs. 0.30 baseline)
  • Weighted F1: 0.76 (vs. 0.66 baseline)

Evaluation metrics

Test accuracy 74.7% · error rate 25.3%. Per-class F1 highlights: akiec 0.53, df 0.47 (baseline 0.00), vasc 0.62 (baseline 0.00), nv 0.86.

Qualitative results

Baseline confusion matrix: very strong on nv (e.g. 965 correct), no correct predictions for df and vasc. Primary model: better minority-class predictions while retaining strong nv performance (~830 correct, ~82% hit rate on that class in the reported matrix).

External evaluation

ISIC 2018 external test set (1,511 images): accuracy 67.3% (~7 points below in-distribution test). Minority-class gains vs. baseline largely held: e.g. class 3 F1 0.39 vs. 0.00, class 6 F1 0.51 vs. 0.00.

Ethical considerations

  • Dataset underrepresents darker skin tones — risk of biased predictions across demographics.
  • Oversampling mitigates class imbalance but does not fix representation gaps.
  • Consent and privacy constraints apply to dermoscopic archives.
  • This is a research / coursework prototype, not a clinical device; it must not replace professional diagnosis.
